GitHub Actions + GHCR

CI/CD + Infrastructure
as Code

Monorepo: FastAPI service, tests with pytest, container image via Docker, publish to GitHub Container Registry, then Ansible (infra/deploy.yml) pulls the image and runs docker compose on staging or production. Prometheus scrapes app metrics and Grafana provides a small monitoring UI. Nginx is the single public entrypoint and routes services by URL path. JSON API: /api · health: /health · metrics: /metrics · dashboards: /grafana/ · Prometheus: /prometheus/

triggers & deploy
push → main
Runs build-and-test, pushes image to GHCR, then deploy-staging (Ansible on self-hosted runner).
workflow_dispatch
Manual run: choose staging or production. Production uses SSH to the VM and GitHub environment gates.
rollback.yml
Manual rollback: choose environment and image tag, then Compose pulls that tag and verifies /health.
runtime
This page is served at /; machine checks use /health; Prometheus uses /metrics.
nginx
Only Nginx is published on the host. It proxies / to FastAPI, /grafana/ to Grafana, and /prometheus/ to Prometheus.
